The workspace relative path to the CONS3RT asset zip file or directory to be used for upload. If the provided path points to a directory, it must contain an asset.properties file at its top level. A zip will be created from the directory and used for asset upload. All CONS3RT assets must contain an asset.properties file specifying information about the asset's structure. More information about creating CONS3RT assets and their asset.properties files can be found here.
